# IT Support Manager & Finance Web Portal

This is the Next.js frontend web application for the IT Support Laptop Lifecycle Management system.

## Getting Started

### Installation
From the `frontend/` directory, install the Node dependencies:
```bash
npm install
```

### Running Locally
To launch the local development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.
